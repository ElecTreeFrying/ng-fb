  FIREBASE RULES
  
    go to firebase console,
    select your project,

    go to database tab in the left pane,
    select rules, copy and past this code
    to allow read and write in
    realtime database without authentication
  
    {
      "rules": {
        ".read": "true",
        ".write": "true"
      }
    }

  
    go to storage tab in the left pane,
    select rules, copy and past this code
    to allow read and write in
    storages without authentication
  
    service firebase.storage {
      match /b/{bucket}/o {
        match /{allPaths=**} {
          allow read, write;
        }
      }
    }
